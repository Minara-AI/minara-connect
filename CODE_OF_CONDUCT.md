# Code of Conduct

cc-connect adopts the [**Contributor Covenant**, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

The full text is canonical at the link above. Read it before participating in this project's spaces (issues, pull requests, discussions, chat rooms hosted on the protocol itself).

## In short

- Be kind. Assume good faith.
- Critique ideas, not people. Disagreement is welcome; contempt is not.
- This is an inclusive project. We welcome contributors of all backgrounds and experience levels.
- Maintainers may remove, edit, or reject contributions, comments, or accounts that violate the Covenant.

## Reporting

If you experience or witness behavior that violates the Covenant, please report it to the maintainers privately:

- Email: **conduct@minara.ai** (or the email listed on the [GitHub organization page](https://github.com/Minara-AI) if that ever changes)
- Or use [GitHub's private contact form](https://github.com/Minara-AI/cc-connect/security/advisories/new) for sensitive matters; the security advisory channel is monitored by the same people.

Reports are handled in confidence. Maintainers will acknowledge within 72 hours and follow up with next steps.

## Scope

The Covenant applies to all project spaces — this repository, its issue tracker, pull requests, project discussions, and any cc-connect-hosted Room used for project coordination. It also applies when an individual is officially representing the project in public spaces.
